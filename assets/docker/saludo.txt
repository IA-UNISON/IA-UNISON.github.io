Este es el contenedor Docker del curso de
Procesamiento de Lenguaje Natural.

Esperamos que en este contenedor ya se encuentren
instaladas todas las dependencias necesarias tanto
para realizar las prácticas del curso, como para
poder realizar otro tipo de proyectos de PLN.

Si necesitas algún paquete extra, lo puedes instalar
utilizando el comando:

$ apt-get install NOMBRE_PAQUETE

Si necesitas algún modulo de python no instalado

$ pip3 install MODULO_PYTHON

Para ejecutar Jupyter

$ run_notebook

y en el navegador normal ir a la dirección http://localhost:8080
